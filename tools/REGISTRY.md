# Marketing Tools Registry （マーケティングツールレジストリ）

Quick reference for AI agents to discover tool capabilities and integration methods. （AI エージェントがツールの機能と統合方法を発見するためのクイック リファレンス。）

## How to Use This Registry （このレジストリの使用方法）

1. **Find tools by category** - Browse sections below for tools in each domain （**カテゴリ別にツールを検索** - 各ドメインのツールについては、以下のセクションを参照してください）
2. **Check integration methods** - See what APIs, MCPs, CLIs, or SDKs are available （**統合方法を確認してください** - 利用可能な API、MCP、CLI、または SDK を確認してください）
3. **Read integration guides** - Detailed setup and common operations in `integrations/` （**統合ガイドを読む** - `integrations/` での詳細なセットアップと一般的な操作）

## By Category （カテゴリ別）

### Analytics （分析）

Track user behavior, measure conversions, and analyze marketing performance. （ユーザーの行動を追跡し、コンバージョンを測定し、マーケティングのパフォーマンスを分析します。）

| Tool（道具） | Best For（最適な用途） | MCP Available（MCPが利用可能） |
|------|----------|:-------------:|
| **ga4**（**が4**） | Web analytics, Google ecosystem（ウェブ解析、Google エコシステム） | ✓ |
| **mixpanel**（**ミックスパネル**） | Product analytics, event tracking（製品分析、イベント追跡） | - |
| **amplitude**（**振幅**） | Product analytics, cohort analysis（製品分析、コホート分析） | - |
| **posthog**（**ポストホッグ**） | Open-source analytics, session replay（オープンソース分析、セッションリプレイ） | - |
| **segment**（**セグメント**） | Customer data platform, routing（顧客データ プラットフォーム、ルーティング） | - |
| **adobe-analytics**（**アドビアナリティクス**） | Enterprise analytics（エンタープライズ分析） | - |

**Agent recommendation**: Start with GA4 if using Google ecosystem. Use Mixpanel or Amplitude for deeper product analytics. （**エージェントの推奨事項**: Google エコシステムを使用している場合は、GA4 から始めてください。より詳細な製品分析には、Mixpanel または Amplitude を使用します。）

### SEO （SEO）

Search engine optimization tools for keyword research, rank tracking, and site audits. （キーワード調査、ランク追跡、サイト監査のための検索エンジン最適化ツール。）

| Tool（道具） | Best For（最適な用途） | Notes（注意事項） |
|------|----------|-------|
| **google-search-console**（**グーグル検索コンソール**） | Free, authoritative search data（無料で信頼できる検索データ） | Direct from Google（Google から直接） |
| **semrush**（**セムラッシュ**） | Competitive analysis, keyword research（競合分析、キーワード調査） | Comprehensive（包括的な） |
| **ahrefs**（**アーレフ**） | Backlink analysis, content research（バックリンク分析、コンテンツ調査） | Best for links（リンクに最適） |

**Agent recommendation**: Google Search Console is essential (free). Add Semrush or Ahrefs for competitive research. （**エージェントの推奨事項**: Google Search Console は必須です (無料)。競合調査のために Semrush または Ahrefs を追加します。）

**Agent recommendation**: HubSpot for startups/SMBs, Salesforce for enterprise. （**エージェントの推奨事項**: スタートアップ/SMB には HubSpot、エンタープライズには Salesforce。）

## MCP-Enabled Tools （MCP 対応ツール）

These tools have Model Context Protocol servers available, enabling direct agent interaction: （これらのツールにはモデル コンテキスト プロトコル サーバーが用意されており、エージェントとの直接対話が可能になります。）

- **ga4** - Google Analytics 4 data access （**ga4** - Google アナリティクス 4 データ アクセス）
- **stripe** - Payment and subscription management （**ストライプ** - 支払いとサブスクリプションの管理）
- **mailchimp** - Email campaign management （**mailchimp** - 電子メール キャンペーン管理）
- **google-ads** - Ad campaign management （**google-ads** - 広告キャンペーン管理）
- **resend** - Transactional email sending （**再送信** - トランザクション電子メールの送信）
- **zapier** - Workflow automation （**zapier** - ワークフローの自動化）

To use MCP tools, ensure the appropriate MCP server is configured in your environment. （MCP ツールを使用するには、環境内に適切な MCP サーバーが構成されていることを確認してください。）
