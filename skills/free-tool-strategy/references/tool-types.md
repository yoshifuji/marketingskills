# Free Tool Types Reference （無料のツール タイプのリファレンス）

Detailed guide to each type of marketing tool you can build. （構築できる各タイプのマーケティング ツールに関する詳細なガイド。）

## Calculators （電卓）

**Best for**: Decisions involving numbers, comparisons, estimates （**最適な用途**: 数値、比較、推定を含む意思決定）

**Examples**: （**例**:）
- ROI calculator （ROI 計算ツール）
- Savings calculator （節約計算機）
- Cost comparison tool （コスト比較ツール）
- Salary calculator （給与計算機）
- Tax estimator （税金見積りツール）
- Pricing estimator （価格見積りツール）
- Compound interest calculator （複利計算ツール）
- Break-even calculator （損益分岐点計算ツール）

**Why they work**: （**機能する理由**:）
- Personalized output （パーソナライズされた出力）
- High perceived value （高い知覚価値）
- Share-worthy results （共有する価値のある結果）
- Clear problem → solution （明確な問題→解決策）

**Implementation tips**: （**実装のヒント**:）
- Keep inputs simple （入力をシンプルにする）
- Show calculations transparently （計算を透過的に表示する）
- Make results shareable （結果を共有可能にする）
- Add "powered by" branding （「powered by」ブランドを追加する）

---

## Generators （発電機）

**Best for**: Creating something useful quickly （**最適な用途**: 役立つものを素早く作成する）

**Examples**: （**例**:）
- Policy generator (privacy, terms) （ポリシー生成者 (プライバシー、規約)）
- Template generator （テンプレートジェネレーター）
- Name/tagline generator （名前/キャッチフレーズジェネレーター）
- Email subject line generator （電子メールの件名ジェネレーター）
- Resume builder （履歴書ビルダー）
- Color palette generator （カラーパレットジェネレーター）
- Logo maker （ロゴメーカー）
- Contract generator （契約生成者）

**Why they work**: （**機能する理由**:）
- Tangible output （目に見えるアウトプット）
- Saves time （時間を節約します）
- Easily shared （簡単に共有）
- Repeat usage （繰り返し使用）

**Implementation tips**: （**実装のヒント**:）
- Output should be immediately usable （出力はすぐに使用できるようにする必要があります）
- Allow customization （カスタマイズを許可する）
- Offer download/export options （ダウンロード/エクスポートのオプションを提供する）
- Include email gating for premium outputs （プレミアム出力用の電子メール ゲーティングを含める）

---

## Analyzers/Auditors （分析者/監査人）

**Best for**: Evaluating existing work or assets （**最適な用途**: 既存の作業または資産の評価）

**Examples**: （**例**:）
- Website grader （ウェブサイトグレーダー）
- SEO analyzer （SEOアナライザー）
- Email subject tester （電子メールの件名テスター）
- Headline analyzer （ヘッドラインアナライザー）
- Security checker （セキュリティチェッカー）
- Performance auditor （業績監査人）
- Accessibility checker （アクセシビリティチェッカー）
- Code quality analyzer （コード品質アナライザー）

**Why they work**: （**機能する理由**:）
- Curiosity-driven （好奇心によって動かされる）
- Personalized insights （パーソナライズされた洞察）
- Creates awareness of problems （問題意識を生み出す）
- Natural lead to solution （Error 500 (Server Error)!!1500.That’s an error.There was an error. Please try again later.That’s all we know.）

**Implementation tips**: （**実装のヒント**:）
- Score or grade for gamification （ゲーミフィケーションのスコアまたはグレード）
- Benchmark against averages （平均に対するベンチマーク）
- Provide actionable recommendations （実用的な推奨事項を提供する）
- Follow up with improvement offers （改善提案をフォローアップする）

---

## Testers/Validators （テスター/バリデーター）

**Best for**: Checking if something works （**最適な用途**: 何かが機能するかどうかを確認する）

**Examples**: （**例**:）
- Meta tag preview （メタタグのプレビュー）
- Email rendering test （電子メールのレンダリングテスト）
- Mobile-friendly test （モバイルフレンドリーテスト）
- Speed test （スピードテスト）
- DNS checker （DNSチェッカー）
- SSL certificate checker （SSL証明書チェッカー）
- Redirect checker （リダイレクトチェッカー）
- Broken link finder （壊れたリンクファインダー）

**Why they work**: （**機能する理由**:）
- Immediate utility （すぐに使える実用性）
- Bookmark-worthy （ブックマークに値する）
- Repeat usage （繰り返し使用）
- Professional necessity （専門的な必要性）

**Implementation tips**: （**実装のヒント**:）
- Fast results are essential （迅速な結果が不可欠です）
- Show pass/fail clearly （合格/不合格を明確に表示する）
- Provide fix instructions （修正手順を提供する）
- Integrate with your product where relevant （関連する場合は製品と統合します）

---

## Libraries/Resources （ライブラリ/リソース）

**Best for**: Reference material （**最適な用途**: 参考資料）

**Examples**: （**例**:）
- Icon library （アイコンライブラリ）
- Template library （テンプレートライブラリ）
- Code snippet library （コードスニペットライブラリ）
- Example gallery （サンプルギャラリー）
- Industry directory （業界ディレクトリ）
- Resource list （リソースリスト）
- Swipe file collection （スワイプファイルコレクション）
- Font pairing tool （フォントペアリングツール）

**Why they work**: （**機能する理由**:）
- High SEO value （高いSEO価値）
- Ongoing traffic （進行中のトラフィック）
- Establishes authority （権威を確立する）
- Linkable asset （リンク可能なアセット）

**Implementation tips**: （**実装のヒント**:）
- Make searchable/filterable （検索/フィルタリング可能にする）
- Allow easy copying/downloading （簡単なコピー/ダウンロードを許可する）
- Update regularly （定期的に更新する）
- Accept community submissions （コミュニティからの投稿を受け入れる）

---

## Interactive Educational （インタラクティブな教育）

**Best for**: Learning/understanding （**最適な用途**: 学習/理解）

**Examples**: （**例**:）
- Interactive tutorials （インタラクティブなチュートリアル）
- Code playgrounds （コードの遊び場）
- Visual explainers （視覚的な説明者）
- Quizzes/assessments （クイズ/評価）
- Simulators （シミュレーター）
- Comparison tools （比較ツール）
- Decision trees （ディシジョンツリー）
- Configurators （コンフィギュレーター）

**Why they work**: （**機能する理由**:）
- Engages deeply （深く関わっていく）
- Demonstrates expertise （専門知識を実証する）
- Shareable （共有可能）
- Memory-creating （記憶の創造）

**Implementation tips**: （**実装のヒント**:）
- Make it hands-on （実践的に行う）
- Show immediate feedback （即時のフィードバックを表示する）
- Lead to deeper resources （より深いリソースにつながる）
- Capture engaged users （エンゲージメントの高いユーザーを獲得する）
