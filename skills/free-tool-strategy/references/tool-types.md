# Free Tool Types Reference （日本語訳：無料のツール タイプのリファレンス）

Detailed guide to each type of marketing tool you can build. （日本語訳：構築できる各タイプのマーケティング ツールに関する詳細なガイド。）

## Calculators （日本語訳：電卓）

**Best for**: Decisions involving numbers, comparisons, estimates （日本語訳：**最適な用途**: 数値、比較、推定を含む意思決定）

**Examples**: （日本語訳：**例**:）
- ROI calculator （日本語訳：ROI 計算ツール）
- Savings calculator （日本語訳：節約計算機）
- Cost comparison tool （日本語訳：コスト比較ツール）
- Salary calculator （日本語訳：給与計算機）
- Tax estimator （日本語訳：税金見積りツール）
- Pricing estimator （日本語訳：価格見積りツール）
- Compound interest calculator （日本語訳：複利計算ツール）
- Break-even calculator （日本語訳：損益分岐点計算ツール）

**Why they work**: （日本語訳：**機能する理由**:）
- Personalized output （日本語訳：パーソナライズされた出力）
- High perceived value （日本語訳：高い知覚価値）
- Share-worthy results （日本語訳：共有する価値のある結果）
- Clear problem → solution （日本語訳：明確な問題→解決策）

**Implementation tips**: （日本語訳：**実装のヒント**:）
- Keep inputs simple （日本語訳：入力をシンプルにする）
- Show calculations transparently （日本語訳：計算を透過的に表示する）
- Make results shareable （日本語訳：結果を共有可能にする）
- Add "powered by" branding （日本語訳：「powered by」ブランドを追加する）

- Template generator （日本語訳：テンプレートジェネレーター）
- Name/tagline generator （日本語訳：名前/キャッチフレーズジェネレーター）
- Email subject line generator （日本語訳：電子メールの件名ジェネレーター）
- Resume builder （日本語訳：履歴書ビルダー）
- Color palette generator （日本語訳：カラーパレットジェネレーター）
- Logo maker （日本語訳：ロゴメーカー）
- Contract generator （日本語訳：契約生成者）

**Why they work**: （日本語訳：**機能する理由**:）
- Tangible output （日本語訳：目に見えるアウトプット）
- Saves time （日本語訳：時間を節約します）
- Easily shared （日本語訳：簡単に共有）
- Repeat usage （日本語訳：繰り返し使用）

**Implementation tips**: （日本語訳：**実装のヒント**:）

- Allow customization （日本語訳：カスタマイズを許可する）
